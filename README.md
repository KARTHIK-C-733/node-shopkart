# A shopping cart project developed with Node JS.

## Development tools

1. Node JS(Server)
2. Bootstrap(CSS/JS)
3. Stripe(Payments)
4. Mongo(Datastore)
5. Handlebars(Templating)
6. Moongoose(ORM)
7. User Authentication(Google Recaptcha, Node Passport)
